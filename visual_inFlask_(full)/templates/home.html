Welcome



Welcome to Where's My Repo, your comprehensive data mining and analysis tool for open-source software projects. Where's My Repo is designed to provide you with invaluable insights into the world of open-source software development.



Our tool mines data from various open-source repositories, including GitHub, to offer aggregated views and interactive dashboards that showcase the latest trends and activities in the open-source community.



Key Features



  • Real-time Data Mining : WMR fetches realtime data from live public repositories, ensuring you have access to the latest information.

  • Modular Design : Our core engine is designed to be flexible, allowing seamless integration of new data sources and APIs.

  • Web-Based Visualization : Explore project metrics, top contributors, commits,and more through our user-friendly web-based visualization tool.

  • Customization : Tailor WMR to your needs by adding new keywords and repositories
